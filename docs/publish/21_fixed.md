# fixed-layout

고정형 레이아웃이란 고정된 너비를 가지는 레이아웃입니다.

예를들어 문서전체의 크기가 1200px 가지는 레이아웃인 경우 아래처럼 지정하면 됩니다.

```css
body {
    width: 1200px;
}
```


레이아웃은 여러가지 형태의 단을 가질 수 있습니다.

예를들어 1단, 2단, 3단 레이아웃이 가장 많이 사용됩니다.
